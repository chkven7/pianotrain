Pianotrain

sight reading piano learning

View on GitHub Download .zip Download .tar.gz

Este software esta indicado para todos aquellos musicos que deseen mejorar sus aptitudes de lectura a primera vista (sightreading) , solfeo cantado (singreading) y entrenamiento de oido (ear training).

El objetivo del programa es conseguir que el usuario adquiera capacidades de lectura a primera vista (sightreading) que le permitan interpretar una partitura a piano con fluidez.

Al mismo tiempo puede mejorar las capacidades de solfeo cantado a primera vista, gracias a un detector de pitch, y de solfeo hablado ya que include un reconocedor de voz .

EL programa tiene aspectos teoricos como circulo de quintas, formacion de escalas y composicion de acordes.

Posee entrada de micro, teclado virtual y midi, capacidad de entreno para cada mano independientemente, medidas de ritmo a nivel de compas y de nota con toleracia ajustable a las necesidades de cada usuario.

Posibilidad de digitacion automatica por medio de un algoritmo genetico.

Generacion de Partituras aleatorias para que el usuario no use la memoria mecanica y simpre tenga que leer. Distintos modos de generacion aleatoria: progresiva y total.

Es el unico programa del mercado que reune caracteristicas de aprendizaje tanto para SightReading como SingReading. La mayoria de programas solo permiten una de las dos y ademas es el unico especializado en generacion aleatoria de partituras.

Este programa esta dirigido a musicos con o sin experiencia que quieran adquirir o mejorar tanto SightReading como SingReading.
